# 📈 FinSight AI

An AI-powered financial research assistant that combines live market data, technical analysis, financial statements, and an LLM to generate professional equity research reports.

## Features

- 📈 Live stock market data using Yahoo Finance
- 📊 Candlestick charts with SMA20, SMA50 and EMA20
- 📑 Income Statement, Balance Sheet and Cash Flow analysis
- 🤖 AI-generated investment analysis using Groq Llama 3.3 70B
- 📉 Trading volume visualization
- 💹 Key financial metrics dashboard

## Tech Stack

- Python
- Streamlit
- Plotly
- yFinance
- Pandas
- Groq API
- Llama 3.3 70B

## Installation

```bash
git clone <repo-url>
cd FinSight-AI
pip install -r requirements.txt
streamlit run app.py
```
